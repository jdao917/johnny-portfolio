Dev Environment Setup Guide

Sample onboarding document showing how a new developer could set up a local automation environment.

Public-safe sample: This is mock documentation only. It does not include real company systems, private URLs, credentials, contacts, or internal steps.

Basic Setup Steps

  • Install Git and confirm git --version.
  • Install Python 3.11+ and confirm python --version.
  • Install VS Code or another approved editor.
  • Clone the sample repository into a local development folder.
  • Create a virtual environment using python -m venv .venv.
  • Activate the environment and install dependencies from requirements.txt.
  • Run a safe local validation command before making changes.
